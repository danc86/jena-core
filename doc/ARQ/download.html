CVS

The complete Eclipse project for ARQ is available via CVS. ARQ is in active development. You may wish to try the version in CVS:

Jena CVS Repository :pserver:anonymous@cvs.sourceforge.net:/cvsroot/jena
module name ARQ.

cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/jena login
cvs -z3 -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/jena co ARQ
cd ARQ
ant jar

There is no password necessary for logging in as anonymous.
